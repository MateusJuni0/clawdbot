## Visão de Deus (Projeto App Despensa/Estoque)
- **Expansão de Escopo:** O humano solicitou um upgrade massivo na funcionalidade "Visão de Deus".
- **Interface:** Exigência de UI estilo iOS (Apple Design), ícones arredondados, glassmorphism. Nada quadrado/feio.
- **IA Visual + Web Search:** O usuário envia foto (ex: vinho), a IA identifica e busca na web onde comprar e preços.
- **Agente Gestor:** Um sub-agente para gerir o estoque conversando com o usuário.
- **Integração Delivery:** Conexão com iFood/Bolt Food para reposição automática.
- **B2B Direto:** Opção de "Robô fala com Fornecedor" para reposição direta, com toggle de ativação.
